安装数据库MySQL

发表于:10月 7 2014, 更新于:10月 7 2014, By taevas
大纲

打开网站下载:http://sourceforge.net/projects/mysql-python/files/latest/download
安装

$tar xfz MySQL-python-1.2.4b4.tar.gz
$cd MySQL-python-1.2.4b4
$sudo python setup.py install

其中执行 sudo python setup.py install 出现问题

EnvironmentError:mysql config not found

解决方法

sudo apt-get install mysql-server

然后又重新执行 sudo python setup.py install 出现问题

error: Setup script exited with error: command 'gcc' failed with exit status 1

通过安装如下库即可解决：

sudo apt-get install python-dev

测试安装是否完成

$python
>>> import MySQLdb
>>>
